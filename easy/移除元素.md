题目：移除元素
题库题号：27
要求：给定一个数组 nums 和一个值 val，你需要原地移除所有数值等于 val 的元素，返回移除后数组的新长度。
```

  class Solution:
      def removeElement(self, nums: List[int], val: int) -> int:
          long = len(nums)
          if long == 1:
              if nums[0] == val:
                  return 0
              else:
                  return 1
          else:
              val_num = 0
              for i in range(long):
                  if nums[i] == val:
                      for j in range(i+1,long):
                          if nums[j] != val:
                              nums[i],nums[j]= nums[j],nums[i]
                              val_num = 1
                              break
                      if val_num == 1:
                          val_num = long -1 - i
          return long - val_num
  ```
