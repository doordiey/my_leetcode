# 反转链表

## 题目描述：

```
定义一个函数，输入一个链表的头节点，反转该链表并输出反转后链表的头节点。
```

## 我的解答：

- 思路
- 题解
  - 双指针迭代-申请结点，修改箭头方向

### Python

```python
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        cur = head
        pre = None
        while cur:
            next_ = cur.next;
            cur.next = pre
            pre = cur
            cur = next_
        return pre
```
